
# Autonomous Quant Hedge Engine

Automated multi-factor stock scanner that sends alerts to Telegram.

## Telegram Setup

1. Open Telegram
2. Search BotFather
3. Send /newbot
4. Save the BOT TOKEN

Get chat id:
Search userinfobot and press start

## GitHub Setup

1. Upload this project to GitHub
2. Go to Settings → Secrets → Actions
3. Add:

TELEGRAM_BOT_TOKEN  
TELEGRAM_CHAT_ID

## Run

GitHub Actions will run every 3 hours on weekdays.
